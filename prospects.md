# Mode: prospects — Compare Multiple Prospects

Read `modes/_shared.md` and `modes/_profile.md` before starting.

## Trigger
User wants to compare or rank multiple prospects against each other.

## Process

### Step 1: Load Reports
For each named prospect, load their evaluation report from `reports/`. If any don't have reports yet, run `prospect` mode first.

### Step 2: Comparison Table
Produce a side-by-side comparison:

| Company | Sector | Score | Archetype | Package | Est. Value | Win Prob | Recommended Priority |
|---------|--------|-------|-----------|---------|------------|----------|----------------------|

### Step 3: Recommendation
Based on the comparison, recommend which 1–3 prospects to pursue first and why.
Consider: score, win probability, deal size, effort required, strategic value.

### Step 4: Action Plan
For the recommended top prospects, produce a sequenced action plan:
- Who to contact first
- What channel to use
- What message angle to lead with
